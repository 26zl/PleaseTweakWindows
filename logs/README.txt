================================================================================
                        PLEASETWEAKWINDOWS - LOG FILES
================================================================================

This folder contains log files generated by PleaseTweakWindows.

LOG FILES:
----------
PleaseTweakWindows.log           - Main application log (DEBUG for app, INFO for libraries)
PleaseTweakWindows-error.log     - Error-only log (ERROR level only)
PleaseTweakWindows-telemetry.log - Action telemetry (script execution tracking)

These logs are automatically created when you run the application and are
useful for troubleshooting if something goes wrong.

LOG RETENTION:
--------------
- Daily logs are kept for 30 days
- Error logs are kept for 90 days
- Total size is capped at 500MB

USAGE:
------
If you encounter any issues:
1. Check PleaseTweakWindows-error.log first for critical errors
2. Review PleaseTweakWindows.log for detailed execution traces
3. Include relevant log excerpts when reporting issues

Note: You can safely delete old log files if needed. New ones will be
created automatically on the next run.

================================================================================
