================================================
PleaseTweakWindows - Scripts Directory
================================================

This folder contains Windows optimization scripts used by PleaseTweakWindows.

IMPORTANT NOTES:
- These scripts modify Windows registry and system settings
- Always create a restore point before applying tweaks
- Edit at your own risk - incorrect modifications can cause system instability

SCRIPT CATEGORIES:
- All windows settings optimized/  - General Windows performance tweaks
- Bcdedit tweaks/                  - Boot configuration optimizations
- Gaming optimizations/            - Gaming-specific performance tweaks
- General Tweaks/                  - Miscellaneous system optimizations
- Network optimizations/           - Network stack tweaks
- Services disable and revert/    - Windows services management
- create_restore_point.ps1         - System restore point creation

USAGE:
- Use the GUI application (PleaseTweakWindows.exe) or
- Use the PowerShell interface (PleaseTweakWindowsPScript.ps1)
- DO NOT run scripts directly unless you know what you're doing

Each tweak has both an "Apply" and "Revert" script to undo changes if needed.

For support: https://github.com/26zl/PleaseTweakWindows/issues
