# Security Policy

## Supported versions

Only the latest published release receives security fixes.

## Reporting a vulnerability

Please do not open a public issue for a vulnerability. Use GitHub's
[private vulnerability reporting](https://github.com/26zl/PleaseTweakWindows/security/advisories/new)
and include:

- affected version and Windows build;
- reproduction steps and expected impact;
- relevant logs with personal information removed.

Do not test against systems you do not own or administer. Reports concerning privileged script
execution, download verification, unsafe reverts, or changes that can prevent Windows from booting
are especially useful.
